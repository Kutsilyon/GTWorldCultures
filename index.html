Basic Info for HTML

There are many things in HTML code. HTML code uses things call tags which need to begin in < and end in >. Those are called angle brackets.
So for example, a basic tag will look like this < h1 >
(just don't put any spaces between the angle brackets). There is a slash there since that tag is used for making a title for your webpage so it needs a slash to tell the computer to end the text here so the code won't be in the actual webpage.

Here are some basic tags

  • < h1 > to < h6 > *
  • < p > *
  • < br >
  • < strong > *
  • < em > *

*-ends with a slash
